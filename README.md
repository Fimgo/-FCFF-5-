# Тема 6: Прогнозування FCFF компанії на 5 років

## Що зроблено

У цій папці підготовлено Excel-модель для розрахункового завдання за темою 6:

- отримано початкові показники компанії за останній рік;
- використано прогнозні припущення для варіанта 4;
- побудовано модель FCFF на 5 років;
- додано графік прогнозного FCFF;
- розрахунки автоматизовано за допомогою Python, щоб уникнути помилок у формулах Excel.

## Файли

- `FCFF_варіант_4_готово.xlsx` - готова Excel-модель для здачі.
- `build_fcff_variant_4.py` - Python-скрипт, який генерує Excel-файл.
- `README.md` - короткий опис завдання, логіки розрахунку та структури файлів.

## Логіка прогнозу

Прогноз побудовано на 5 років. Для кожного прогнозного року використовуються такі формули:

```text
Виручка t = Виручка t-1 × (1 + темп росту виручки)

Операційні витрати t = Виручка t × частка операційних витрат у базовому році

Амортизація t = Виручка t × частка амортизації у базовому році

EBIT = Виручка - Операційні витрати - Амортизація

Податки = EBIT × Податкова ставка

EBIT (1 - Tax) = EBIT - Податки

CapEx t = Виручка t × CapEx як % від виручки

ΔWC t = (Виручка t - Виручка t-1) × ΔWC як % від приросту виручки

FCFF = EBIT (1 - Tax) + Амортизація - CapEx - ΔWC
```
